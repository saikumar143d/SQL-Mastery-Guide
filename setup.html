Option 2: Local Database Setup (The "Real World" Approach)

Installing a database on your computer gives you full control and mimics a professional development environment. For learning, you can't go wrong with these free and powerful options.

data_objectPostgreSQL + DBeaver

This combination provides a powerful, professional-grade environment that works on Windows, macOS, and Linux.

Step 1: Install PostgreSQL

Go to the official PostgreSQL download page and run the installer for your OS. During setup, you'll be asked to set a password for the `postgres` superuser. Remember this password!

Step 2: Install DBeaver

Go to the DBeaver download page and install the Community Edition. Open it and click the "New Database Connection" icon. Select PostgreSQL and enter your details (Host: `localhost`, Port: `5432`, User/DB: `postgres`, and your password). Test the connection and you're ready!

storageMySQL + Workbench

Download the official MySQL Installer, which bundles the server and the Workbench GUI.

Step 1: Download & Install

Go to the MySQL Community Downloads page, get the "MySQL Installer for Windows", and run it. Choose the Developer Default setup type.

Step 2: Configure Server

During configuration, you must set a password for the root user. This is critical. Keep other settings at their defaults.

Step 3: Connect

Open MySQL Workbench. Click the "Local instance" connection and enter the root password you created to connect to your server.

terminalSQLite

SQLite is serverless and zero-configuration. The entire database is a single file, making it perfect for beginners.

Installation

Windows: Download binaries from the SQLite website. macOS: Run `brew install sqlite`. Linux: Run `sudo apt-get install sqlite3`. Verify with `sqlite3 --version`.

Loading Data

To create a database from a script (e.g., `hr_database.sql`), run this command in your terminal:

sqlite3 hr.db ".read hr_database.sql"

You can then access it with `sqlite3 hr.db`.
